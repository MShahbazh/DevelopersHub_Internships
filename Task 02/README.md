# Predict Future Stock Prices (Short-Term)  
 
## TASK OBJECTIVE :   
Use historical stock data to predict the next day's closing price.  

## Dataset Used :  
yfinance Tesla stock prices    

## Algorithms Applied :  
RandomForestRegressor and LinearRegression  

## Key results and findings :  
Accuracy LinearRegression : SCORE : 0.9987483649224839  
Accuracy RandomForestRegressor : SCORE : 0.9967407869742254  
LinearRegression performed slightly better than RandomForestRegressor. Overall both ALgorithms performed well.
