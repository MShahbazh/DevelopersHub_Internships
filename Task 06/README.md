# House Price Prediction    

## TASK OBJECTIVE :   
Predict house prices using property features such as size, bedrooms, and location.          
     
## Dataset Used :  
House Price Prediction Dataset    

## Algorithms Applied :  
GradientBoostingRegressor and LinearRegression  

## Key results and findings :  
Accuracy GradientBoostingRegressor : -0.0350823587216349      
Accuracy LinearRegression : -0.008143099974382118      
GradientBoostingRegressor performed slightly better than LinearRegression. Overall both ALgorithms performed worst. This is mainly due to feature selection
