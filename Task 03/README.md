# Heart Disease Prediction  

## TASK OBJECTIVE :   
Build a model to predict whether a person is at risk of heart disease based on their health data.    

## Dataset Used :  
Heart Disease UCI Dataset     

## Algorithms Applied :  
DecisionTreeClassifier and LogisticRegression  

## Key results and findings :  
Accuracy DecisionTreeClassifier : SCORE : 0.2750877192982456    
Accuracy LogisticRegression : SCORE : 0.3508771929824561    
LogisticRegression performed slightly better than DecisionTreeClassifier. Overall both ALgorithms performed worst.
